Setting up Bluebook

A quick, one-time setup so your child is ready for their College Board practice test. Please complete this at home before the session — it saves class time and avoids Wi-Fi and update issues on test day.

Before class

Install Bluebook, open it once, and confirm the student can log in — at home, ahead of the session. Bring that same device fully charged, with the charger if possible.

Links

Important links

  • Create a College Board student accountaccount.collegeboard.org/login/signUp
  • Download the Bluebook appbluebook.collegeboard.org/students/download-bluebook
  • College Board help, if neededsupport.collegeboard.org
Setup

Option A — Student already has a College Board account

  1. Install the Bluebook app using the download link above.
  2. Open the Bluebook app on the laptop or iPad the student will use for the practice test.
  3. Click Sign In / Log In.
  4. Enter the student's College Board username/email and password.
  5. After login, confirm the app opens correctly and there are no update or error messages.
  6. Keep the login details available for test day.
Setup

Option B — Student does not have a College Board account

  1. Go to account.collegeboard.org and start the sign-up.
  2. Choose Student when asked what type of account to create.
  3. Enter the student's real information: full legal name, date of birth, email address, and school information if requested.
  4. Create a username/email and password. Write them down in a safe place.
  5. Check the student/parent email inbox for a verification email and complete the step if requested.
  6. Download and install Bluebook from the download link above.
  7. Open Bluebook and sign in using the newly created College Board student account.
  8. Confirm that Bluebook opens properly before coming to class.
Test Day

Device checklist

  • Bring the same device where Bluebook was installed and tested.
  • Charge the device fully before arriving.
  • Bring the charger if possible.
  • Make sure the student knows their College Board login details.
  • Arrive early enough to connect to Wi-Fi and resolve any small issues.
Good to Know

Notes for parents

  • Please create only one College Board account per student.
  • Use the student's real name and date of birth, so the account can be used later for PSAT/SAT activities.
  • Bluebook does not work on phones. Personal Chromebooks are generally not supported unless they are school-managed.
  • Completing this setup at home saves class time and reduces Wi-Fi and update issues during the session.

Trouble logging in or installing? Reach out to your child's tutor or the CPP team before test day — we're glad to help you get set up.
